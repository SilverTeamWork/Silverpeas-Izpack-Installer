Démarrer Silverpeas

Maintenant que l'installation est terminée, voici les instructions pour jouer avec Silverpeas :

  • Démarrer Silverpeas : ${INSTALL_PATH}\bin\silverpeas.bat start
  • Arrêter Silverpeas : ${INSTALL_PATH}\bin\silverpeas.bat stop
Il ne reste plus qu'à vous connecter sur Silverpeas http://localhost:8000/silverpeas avec le compte: SilverAdmin/SilverAdmin.
