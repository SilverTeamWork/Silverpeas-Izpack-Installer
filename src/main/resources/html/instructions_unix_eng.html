Launching Silverpeas

Now that the installation is finished, these are the instructions to launch Silverpeas :

  • Start the database : ${INSTALL_PATH}/bin/start_db.sh
  • Start Silverpeas : ${INSTALL_PATH}/bin/silverpeas_start_jboss.sh
Now you can connect to Silverpeas http://localhost:8000/silverpeas using the following account : SilverAdmin/SilverAdmin.
