Launching Silverpeas

Now that the installation is finished, these are the instructions to play with Silverpeas:

  • Start Silverpeas : ${INSTALL_PATH}\bin\silverpeas.bat start
  • Stop Silverpeas : ${INSTALL_PATH}\bin\silverpeas.bat stop
Now you can connect to Silverpeas http://localhost:8000/silverpeas using the following account: SilverAdmin/SilverAdmin.
