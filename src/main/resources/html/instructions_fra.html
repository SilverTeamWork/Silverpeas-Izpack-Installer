Démarrer/Arrêter Silverpeas

Maintenant que l'installation est terminée, pour jouer avec Silverpeas allez dans le dossier bin du répertoire ${INSTALL_PATH}, puis  :

  • Pour démarrer Silverpeas : silverpeas start
  • Pour arrêter Silverpeas  : silverpeas stop

Accéder à Silverpeas

Silverpeas ayant été démarré automatiquement suite à cette installation, il ne reste plus qu'à vous connecter sur Silverpeas à http://localhost:8000/silverpeas avec le compte : SilverAdmin/SilverAdmin. Le mot de passe des autres comptes pré-enregistrés est sasa.
