Démarrer Silverpeas

Maintenant que l'installation est terminée, voici les instructions pour jouer avec Silverpeas. Allez dans le dossier bin du répertoire ${INSTALL_PATH}, puis :

  • Démarrer Silverpeas : silverpeas start
  • Arrêter Silverpeas : silverpeas stop

Il ne reste plus qu'à vous connecter sur Silverpeas http://localhost:8000/silverpeas avec le compte: SilverAdmin/SilverAdmin.
