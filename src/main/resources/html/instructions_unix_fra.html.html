Démarrer Silverpeas

Maintenant que l'installation est terminée, pour lancer Silverpeas vous devez effectuer les opérations suivantes :

  • Démarrer la base de donnée : ${INSTALL_DIR}\bin\start_db.bat
  • Démarrer Silverpeas : ${INSTALL_DIR}\bin\silverpeas_start_jboss.cmd
Il ne reste plus qu'à vous connecter sur Silverpeas http://localhost:8000/silverpeas avec le compte: SilverAdmin/SilverAdmin.
