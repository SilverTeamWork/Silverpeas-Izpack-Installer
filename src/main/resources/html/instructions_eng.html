Launching Silverpeas

Now that the installation is finished, these are the instructions to play with Silverpeas. Go to the bin directory in ${INSTALL_PATH} and then:

  • Start Silverpeas : silverpeas start
  • Stop Silverpeas : silverpeas stop

Now you can connect to Silverpeas http://localhost:8000/silverpeas using the following account: SilverAdmin/SilverAdmin.
