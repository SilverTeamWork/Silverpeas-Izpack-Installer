Starting/Stopping Silverpeas

Now that the installation is finished, you can play with Silverpeas by goinf into the bin folder of the ${INSTALL_PATH} directory and then:

  • To start Silverpeas: silverpeas start
  • To stop Silverpeas: silverpeas stop

Connection to Silverpeas

As Silverpeas was been eventually started by this installer, you can connect to Silverpeas at http://localhost:8000/silverpeas using the following account: SilverAdmin/SilverAdmin. The password of the other pre-recorded user accounts is sasa
